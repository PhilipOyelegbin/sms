School Management System API

Routes

User Routes
  • Get and post a user: https://studentmanagementsystem-api.vercel.app/v1/users
  • Get, update and delete a user by email: https://studentmanagementsystem-api.vercel.app/v1/users/:id
  • Authenticate a user: https://studentmanagementsystem-api.vercel.app/auth/users
Student Routes
  • Get and post a student: https://studentmanagementsystem-api.vercel.app/v1/students
  • Get, update and delete a student by email: https://studentmanagementsystem-api.vercel.app/v1/students/:id
  • Authenticate a user: https://studentmanagementsystem-api.vercel.app/auth/students
Score Routes
  • Get and post a score: https://studentmanagementsystem-api.vercel.app/v1/scores
  • Get, update and delete a score by id: https://studentmanagementsystem-api.vercel.app/v1/scores/:id
  • Get all scores regarding a student by student email: https://studentmanagementsystem-api.vercel.app/v1/scores/query?students=email
